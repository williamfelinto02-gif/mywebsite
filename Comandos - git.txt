
Branch

    git branch --- verifica quantas branchs existem e onde está.
    git branch <nome da branch> ou  git switch -c <nome da branch> --- cria uma nova branch
    git switch <nome da branch> --- Altera entre as branch's
    git branch -d <nome da branch> --- deleta a branch


GitchHub

     git push --all  ---- envia todos as alterações para o github
     git pull ----- Cópia do github
